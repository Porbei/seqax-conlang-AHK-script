# seqax-conlang-AHK-script
An Auto Hotkey script for my conlang, Ðélas (You will have to disable windows game bar for this to work properly, or you can download and edit the file to your preferences).

Ash:
Win+t=Þ
Alt+t=þ

Eth:
Ctrl+2=Ð | Alt+2=ð

N tilde:
Win+y=Ñ | Alt+y=ñ

N acute:
Win+g=Ń | Alt+g=ń

C cedilla:
Win+c=Ç | Alt+c=ç

S caron:
